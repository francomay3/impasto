# CanvasEngine Refactor — Agent Comments

This file is maintained by the executing agent across phases. Write here whenever
you encounter something surprising, make a decision that deviates from the plan,
hit an obstacle and find a workaround, or learn something that future phases should
know. Read this file at the start of every phase before touching code.

---

## How to use this file

- **Before starting a phase:** read all existing comments, especially the section
  for the previous phase and any "watch out" entries.
- **After completing a phase:** add a section below with what you learned.
- **If you deviate from the plan:** document the deviation and the reason here.
  Do not silently change the plan — write it down.
- **If you discover a dependency between phases not in the plan:** write it here
  so the next phase doesn't get blindsided.

---

## Format for phase entries

```
### Phase N — [Phase Name] — [status: completed | in progress | blocked]

**Date completed / attempted:**

**Deviations from plan:**
- (none) OR describe what changed and why

**Surprises / obstacles:**
- describe anything unexpected

**Decisions made:**
- any architectural decision made mid-phase

**Watch out in future phases:**
- anything the next agent should know before starting the next phase
```
